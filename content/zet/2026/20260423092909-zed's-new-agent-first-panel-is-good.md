# Zed's new Agent first panel is good

I really like Zed's new Agent first panel. 

Bummer that I just spent time setting up my hotkeys to open the agent panel e.g. `<leader>ccc` - kind of moot now. But, this is a better change!

I still prefer using JetBrains products (especially for debugging) but for pure "agentic" style work (such as project I don't care *that* much about) it's 
a better, faster experience.

There might be a future where I'd switch solely to Zed over JetBrains - but I still think that future is a way off yet.

Love seeing teams try new things like this too - the "agentic" world is in so much flux right now so who knows if they'll even keep this change. I hope they do.
